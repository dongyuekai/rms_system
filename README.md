# 挂号管理系统

## 安装依赖
```bash
yarn install-all
```

## 初始化数据
```bash
cd backend && node scripts/initData.js
```

## 启动项目
```bash
yarn dev
```

## 功能特性
- 实时挂号状态同步 (WebSocket)
- 候补挂号功能
- 移动端适配
- 个人挂号记录管理
